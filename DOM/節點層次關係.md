# 節點層次關係

## 節點
- 通過節點的層次關係獲取節點對象。
- 關係:
1. parentNode(父節點): 對應一個節點對象。
2. childNodes(子節點): 對應一個節點集合。
3. 兄弟節點:
   上一個兄弟節點: previousSibling
   下一個兄弟節點: nextSibling

			//獲取頁面中的表格節點
			var tabNode = document.getElementById("tabid");

			//獲取父節點 -> parentNode
			// var node = tabNode.parentNode;

			//獲取子節點 -> childNode
			// var nodes = tabNode.childNodes;
